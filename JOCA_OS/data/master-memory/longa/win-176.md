# Tentativa de fechar terminal UniMedia no JOCA UI

TAGS: joca-os, unimedia, workers, terminal-ui, limitações-master
JANELA: win-176 (8 turnos)

Renato pediu ao Master JOCA que fechasse o terminal do worker `8bd56af5...` (UniMedia) na UI do JOCA UI.
O worker tinha concluído commit `fd7a4d1` e push para `https://github.com/MirrasPT/UniTV.git` (branch `master`).
Ficheiros `untracked` locais restantes: `.joca/` e `graphify-out/`, fora do commit.
O Master conseguiu encerrar a sessão interna do terminal (shell fechada, worker `idle`), mas não conseguiu remover a janela da UI.
Não existe ferramenta `close_worker` / `close_terminal` exposta ao Master nesta versão do JOCA OS.
A tentativa de controlo via browser também falhou por ausência de browser na sessão.
Worker `87f86326...` (`LE_Plano`) permaneceu em `working` sem intervenção.
Ficou claro que fechar a shell ≠ remover o terminal da lista do JOCA UI — lacuna de funcionalidade a endereçar.
Renato precisou de fechar o terminal manualmente pela própria interface do JOCA UI.
Ponto de melhoria identificado: o Master deve ter acesso a um comando `close_worker`/`remove_terminal` no JOCA OS.
